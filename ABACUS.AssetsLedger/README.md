# ABACUS.AssetsLedger

Ce module couvre la gestion des immobilisations.

Il sert a suivre les biens immobilises, leur classification et les donnees de gestion qui evoluent au fil du temps.

## Ce que couvre le module

- les immobilisations
- les categories d'immobilisation
- les amortissements
- les imputations comptables et analytiques
- les champs libres et autres details dependants
- les types de transactions et les ecritures d'immobilisation
- les soldes et classifications

Ce projet est adapte quand une integration doit piloter le cycle de vie des actifs immobilises dans ABACUS.
