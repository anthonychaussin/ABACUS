# ABACUS.AccountsReceivable

Ce module couvre la comptabilite clients.

Il sert a manipuler les donnees de facturation cote client, depuis les fiches clients jusqu'aux encaissements, relances et autres mouvements associes.

## Ce que couvre le module

- les clients
- les devises clients
- les factures clients
- les positions de facture
- les prelevements directs
- les paiements et remboursements
- les relances et impayes

Ce projet est utile pour les integrations qui suivent le cycle de vente et d'encaissement.
