# ABACUS.Tests

Ce projet regroupe les tests du SDK.

Il ne represente pas un module metier ABACUS. Son role est de verifier que les principaux clients du depot peuvent etre instancies et appeles correctement sur un socle minimal de scenarios.

## Ce que couvre le projet

- des smoke tests sur certains modules
- la verification du comportement de base des clients
- des utilitaires de test pour simuler les appels HTTP

Ce projet sert surtout de garde-fou pour les evolutions du SDK.
