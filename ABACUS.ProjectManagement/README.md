# ABACUS.ProjectManagement

Ce module couvre la gestion de projets.

Il sert a suivre les projets, organiser les equipes, planifier les ressources et gerer les tarifs, imputations et documents lies a l'activite projet.

## Ce que couvre le module

- les projets
- les equipes projet
- la planification du personnel et des assets
- les plannings mensuels
- les codes de prestation et de facturation
- les tarifs par employe, asset, projet, client ou groupe
- les imputations projet
- les documents de projet et de booking

Ce projet est pertinent pour les integrations de pilotage projet, de planification et de facturation de prestations.
