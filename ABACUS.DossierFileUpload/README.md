# ABACUS.DossierFileUpload

Ce module couvre le depot de fichiers dans les dossiers ABACUS.

Il sert a envoyer des documents vers des espaces de stockage et des dossiers lies a differents objets metier.

## Ce que couvre le module

- l'upload de fichiers dans le filestore utilisateur
- l'ajout de documents dans des dossiers de factures clients
- l'acces a certains dossiers de stockage
- des points d'entree lies au depot de contenu sur des objets metier

Ce projet est utile des qu'une integration doit pousser un document dans ABACUS plutot que seulement manipuler des donnees metier.
