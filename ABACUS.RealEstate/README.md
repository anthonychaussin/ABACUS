# ABACUS.RealEstate

Ce module couvre la gestion immobiliere.

Il sert a manipuler la structure d'un patrimoine immobilier, les acteurs qui y sont rattaches et les documents ou contrats associes.

## Ce que couvre le module

- les immeubles et ensembles immobiliers
- les proprietaires
- les locataires
- les batiments, maisons, etages et objets
- les contrats rattaches aux objets
- les documents immobiliers
- les assemblees de copropriete
- les loyers sur chiffre d'affaires

Ce projet est adapte pour les integrations qui suivent la vie d'un parc immobilier et de ses relations contractuelles.
