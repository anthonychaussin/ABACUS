# ABACUS.General

Ce module couvre les referentiels communs.

Il sert a exposer les donnees transverses qui sont reutilisees par plusieurs domaines ABACUS.

## Ce que couvre le module

- les pays
- les devises
- les divisions
- les entreprises
- les taux de change
- les institutions financieres
- les stockages et dossiers communs
- des traitements batch generiques

Ce projet est utile quand une integration a besoin de s'appuyer sur les donnees de reference du systeme.
