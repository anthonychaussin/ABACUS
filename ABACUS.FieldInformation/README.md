# ABACUS.FieldInformation

Ce module couvre les informations de champs et certaines metadonnees metier.

Il sert a recuperer ou publier des informations associees a des champs metier ABACUS, par exemple pour mieux comprendre une structure fonctionnelle ou enrichir une integration.

## Ce que couvre le module

- les informations descriptives de champs
- les points d'entree de metadonnees lies a des domaines fonctionnels

Ce projet a du sens quand une integration a besoin du contexte d'un champ, et pas seulement de sa valeur.
