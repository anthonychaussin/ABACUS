# ABACUS.UserDependentAuth

Ce module couvre des autorisations dependantes de l'utilisateur courant.

Il sert a verifier ou recuperer des informations d'acces sur certains contenus ABACUS dont la visibilite depend du contexte utilisateur.

## Ce que couvre le module

- les autorisations sur des documents projet
- les autorisations sur des espaces de stockage

Ce projet est pertinent quand une integration doit tenir compte des droits reels d'un utilisateur avant d'afficher ou d'exploiter un contenu.
