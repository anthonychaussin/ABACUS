# ABACUS.HumanResources

Ce module couvre les ressources humaines hors paie stricte.

Il sert a travailler sur l'organisation, les parcours collaborateurs, la formation et une partie des processus de recrutement.

## Ce que couvre le module

- l'organisation interne
- les employes, positions, roles et delegations
- la formation et le learning
- les CV et historiques de parcours
- les evenements et documents de formation
- les candidatures, offres et objets de recrutement

Ce projet est pertinent pour les integrations RH orientees organisation, talent et suivi collaborateur.
