# ABACUS.Subscription

Ce module couvre les abonnements aux changements et aux evenements.

Il sert a s'abonner a des sujets exposes par ABACUS, a consommer les messages emis et a les acquitter une fois traites.

## Ce que couvre le module

- la consultation des abonnements
- la souscription a un ou plusieurs sujets
- la desinscription
- la consommation des evenements
- l'acknowledgement des messages
- les synchronisations completes sur certains topics

Ce projet est utile pour les integrations evenementielles ou les mecanismes de synchronisation incrementale.
