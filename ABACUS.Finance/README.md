# ABACUS.Finance

Ce module couvre la finance generale.

Il sert a manipuler les referentiels et mouvements comptables qui structurent la tenue financiere dans ABACUS.

## Ce que couvre le module

- les comptes comptables
- les documents de compte
- les classifications de comptes
- les centres de charges
- les ecritures du grand livre
- les documents lies aux ecritures

Ce projet est pertinent pour les integrations qui alimentent ou consultent la comptabilite generale.
