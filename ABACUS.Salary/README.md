# ABACUS.Salary

Ce module couvre la paie et une grande partie des donnees salariales collaborateur.

Il sert a gerer les informations contractuelles et salariales, les notes de salaire, certains objets d'assurance et plusieurs ensembles de donnees RH relies a la paie.

## Ce que couvre le module

- les notes de salaire et leurs placeholders
- les assurances rattachees aux employes
- les entrees et sorties
- les donnees personnelles et contractuelles
- les documents employe
- les notes RH
- les acces portail
- les classifications employees

Ce projet est pertinent pour les integrations qui touchent a la paie, aux contrats ou au dossier salarial collaborateur.
