# ABACUS.ProductionPlanning

Ce module couvre la planification et le suivi de production.

Il sert a exploiter les ressources de production, leurs couts, ainsi que les donnees d'import et d'export liees aux ordres de fabrication.

## Ce que couvre le module

- les ressources de production
- les taux de cout des ressources
- les champs utilisateur et dossiers des ressources
- les donnees d'export de production
- les donnees d'import de confirmations
- les entetes d'ordres de fabrication

Ce projet est utile pour les integrations qui relient ABACUS a un environnement industriel ou atelier.
