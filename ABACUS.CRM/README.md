# ABACUS.CRM

Ce module couvre la gestion des tiers et des relations.

Il sert a centraliser les sujets metier autour des personnes et des organisations : identite, adresses, moyens de contact, liens et regroupements.

## Ce que couvre le module

- les sujets et tiers
- les adresses
- les donnees de communication
- les personnes de contact et les liens entre entites
- les types de liens
- les regroupements et leurs entrees
- la detection de doublons
- les dossiers rattaches aux sujets et aux liens

Ce projet est pertinent pour les integrations qui alimentent ou exploitent la base de contacts ABACUS.
