# ABACUS.WebShop

Ce module couvre les flux webshop.

Il sert a gerer les comptes acheteurs et a pousser des demandes commerciales issues d'un parcours e-commerce vers ABACUS.

## Ce que couvre le module

- les comptes shoppers
- la creation et la mise a jour de comptes acheteurs
- l'envoi de commandes de vente
- l'envoi d'offres de vente
- l'envoi de retours de vente

Ce projet est utile pour relier une boutique en ligne ou un front de commande a ABACUS.
